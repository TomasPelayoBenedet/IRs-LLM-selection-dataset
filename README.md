# Selected Functions Dataset

This repository contains the dataset of selected functions used in the research paper **"Does Representation Matter? Evaluating IRs for LLM-based Binary Decompilation"**.

## Content

The `selected_functions/selected.json` contains all the functions used in our evaluation. It includes the verified paths and function signatures derived from the [ExeBench](https://huggingface.co/datasets/jordiae/exebench) dataset (test real split).
